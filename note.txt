■テーブル設計

blogs
  id
  title: string

entries
  id
  blog_id: references
  title:   string
  body:    string

comments
  id
  entry_id: references
  body:     string
  approved: bool


> rails g scaffold blog title:string
> rails g scaffold entry blog:references  title:string body:string
> rails g scaffold comment entry:references body:string
